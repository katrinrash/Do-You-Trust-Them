# Do You Trust Them?

It is a 3D FPP detective game with narrative elements, inspired by Papers, Please, where the player takes the role of an office worker tasked with identifying a fraudster within a limited time. The player explores the office environment, interacts with computers, documents, and objects, and collects clues to solve the mystery.

## Used Technologies:
- Unity / C#
- GitLab / GitKraken
- Trello / Miro

## Development Process:
"Do You Trust Them?" was created as a university project by a four-person team over the course of 4 months. I worked as one of two programmers and was responsible for developing gameplay systems, UI logic, and narrative-related mechanics.

## My Contributions:
- Designed and implemented the day system responsible for:
  - time progression,
  - day transitions,
  - triggering events and activities.

- Created a computer hacking mini-game and expanded the office computer system connected to the day cycle.

- Fully designed and implemented the notebook system used for tracking clues and collected information.

- Implemented:
  - multiple endings,
  - an achievement system based on player choices and story outcomes.

- Developed:
  - main menu logic,
  - animated day transition screens,
  - a text notification system reacting to player actions.

- Worked on gameplay flow, UI integration, and overall game feel.

## What I Learned: 
- Designing interconnected gameplay systems
- Understanding scene flow and additive scene loading
- Creating event-driven systems
- Collaborating with another programmer on shared systems and architecture
- Improving code organization and project structure in larger Unity projects
